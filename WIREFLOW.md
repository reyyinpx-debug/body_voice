# Body Voice · Wireframe + Userflow v2

## 一句话
你的身体有自己的社交账号——可穿戴设备数据，AI 以"身体本人"口吻发动态、写签文、聊身体。

---

## 用户流程（共 3 步）

```
[上传 CSV] → [AI 批量生成] → [4 Tab 主页]
    ①                ②             ③
```

### ① 上传页（仅首次）
- 拖拽或选择 Oura CSV 文件
- 或用示例数据一键体验
- 选择语气（守护者 / 懒得伺候者 / 上奏者 / 记录者）
- 点击"开始分析"

### ② 加载页
- 显示进度 + 动态文案
- 全部生成完毕后自动跳转

### ③ 主页（底部 4 Tab）

---

## 页面结构

### 全局框架

```
┌──────────────────────────┐
│  [顶栏：标题 + 天数]      │  ← sticky top
├──────────────────────────┤
│                          │
│    (Tab 内容区)           │
│                          │
├──────────────────────────┤
│ 今日签 │ 动态 │ 对话 │ 我的 │  ← 底部 Tab Bar
└──────────────────────────┘
```

- 顶栏 + 内容区 + 底部 Tab Bar（移动端标准布局）
- Tab Bar sticky bottom，内容区可滚动
- 首次上传流程走完后不再看到上传页
